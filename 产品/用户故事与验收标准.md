# 用户故事与验收标准

> 关联：[[产品/PRD 写作模板|PRD 写作模板]]、[[软件工程/测试与质量保障体系|测试与质量保障体系]]

## 用户故事格式

作为【角色】，我希望【能力】，以便【价值】。

## 验收标准

Given/When/Then：

```gherkin
Given 我是管理员
When 我删除一个项目
Then 系统要求二次确认
And 审计日志记录删除动作
```

## 检查

是否覆盖正常、异常、权限、空状态、边界、移动端、可观测性。
