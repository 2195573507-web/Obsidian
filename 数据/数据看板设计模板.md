# 数据看板设计模板

> 关联：[[数据/指标体系设计|指标体系设计]]

## 结构

目标 → 北极星指标 → 趋势 → 分布 → 漏斗 → 明细 → 异常解释。

## 模板

```markdown
# 看板：名称
## 目标
## 用户
## 指标口径
## 图表
## 筛选器
## 告警阈值
## 数据源
## 刷新频率
```

看板必须能支持决策，不只是展示数字。
