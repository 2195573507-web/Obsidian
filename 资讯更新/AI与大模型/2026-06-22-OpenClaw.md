# AI / 大模型 / Agent 更新

日期：2026-06-22
范围：过去 24 小时
说明：以下内容优先基于官方/原始来源；转载或二手解读已尽量回避。无法直接确认的条目标注为“未确认”。

## 最重要 5 条

1. **Ollama 继续强化“本地模型 + Agent 工具链”定位**  
   官方首页与文档明确把 Ollama 定位为最容易上手的本地大模型运行入口，并突出与 Claude Code、Codex、Copilot、OpenClaw 等集成；这对本地私有化部署和个人自动化很直接有用。来源：Ollama 官网/文档。

2. **Ollama 模型库持续更新，面向新模型的接入仍在推进**  
   GitHub 与模型库页面显示其持续支持 Kimi-K2.6、GLM-5.1、gpt-oss、Qwen、Gemma 等模型，适合跟踪“可本地跑的最新开源/开放权重模型”。来源：Ollama GitHub / 模型库。

3. **微软亚洲研究院开源 Agent Lightning**  
   该项目主打“训练-智能体分离式架构”，让任意 LLM Agent 以低侵入方式接入强化学习训练，强调统一轨迹接口与分层 RL；对 Agent 研发、在线反馈优化和自我进化很有参考价值。来源：Microsoft Research。

4. **百度 ERNIE 4.5 系列正式开源**  
   官方博客称开源 10 款模型，覆盖 MoE 与稠密模型，并提供 Paddle/FastDeploy 生态支持、OpenAI 兼容服务部署路径；如果要做国产模型本地/私有化部署，这是值得关注的备选。来源：ERNIE 官方博客。

5. **未确认：多个二手渠道声称有新一代开源模型/Agent 大版本发布**  
   搜索结果里出现 Kimi K2.6、Gemma 4、DeepSeek-V4 等说法，但我没拿到同等级别的原始发布页佐证，暂不当作已确认事实。建议后续再核验官方公告或仓库 release。

## 对 OpenClaw / Hermes / 个人服务器有用的 3 条

1. **Ollama 的 OpenClaw 集成说明很直接**  
   官方文档明确提到可用 `ollama launch openclaw` 把 Ollama 接到个人助手工作流，这对本地模型 + 自动化任务链是现成入口。

2. **Agent Lightning 适合做“可训练的 Agent 层”**  
   如果你们的 OpenClaw/Hermes 未来要做任务成功率优化、轨迹回放、奖励学习，Agent Lightning 的“分离式训练”思路很值得复用。

3. **ERNIE 4.5 + FastDeploy 适合作为私有化服务备选**  
   官方已给出 OpenAI 兼容 API 服务部署示例，适合放到个人服务器上做模型网关/实验环境。

## 风险 / 机会

- **机会**：本地模型生态在继续向“Agent 化、工具化、可训练”演进，Ollama 这类入口对个人服务器价值上升。  
- **机会**：如果 OpenClaw 要增强稳定性，统一轨迹、统一 API、OpenAI 兼容服务这些方向都在变成事实标准。  
- **风险**：大量中文二手转载夹带夸张标题，容易把“传闻版发布”当事实；今天这批结果里尤其明显。
