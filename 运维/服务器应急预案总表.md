# 服务器应急预案总表

> 关联：[[运维/服务器运维 MOC|服务器运维 MOC]]

| 事故 | 首查 | 止血 | 风险 |
|---|---|---|---|
| 全站不可用 | Nginx、UFW、DNS | 恢复入口/回滚配置 | 高 |
| 502 | Nginx error、后端端口 | 重启单后端/回滚 | 中高 |
| 磁盘满 | df、du、docker df | 清理确认日志/缓存 | 高 |
| SSH 异常 | auth log、last | 保留证据、加固 | 高 |
| 证书过期 | certbot、openssl | 续期、reload | 中 |
| DB 异常 | logs、连接、磁盘 | 恢复备份/只读模式 | 高 |

## 原则

止血优先，保留证据，不做无关大改，所有危险操作记录时间线。
