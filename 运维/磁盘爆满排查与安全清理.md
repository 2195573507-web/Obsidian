# 磁盘爆满排查与安全清理

> 关联：[[运维/Linux 系统管理与故障排查 SOP|Linux 系统管理与故障排查 SOP]]

## 1. 先定位

```bash
df -h
df -ih
du -xhd1 / 2>/dev/null | sort -h
du -xhd1 /var 2>/dev/null | sort -h
docker system df
```

## 2. 相对安全清理

journal vacuum、apt cache、临时构建目录、确认无用日志压缩包。

## 3. 高风险

Docker volume、数据库目录、上传文件、备份目录、/var/lib/docker、未知大文件。删除前必须确认用途和备份。

## 4. 防复发

日志轮转、Docker logging max-size、磁盘阈值告警、备份保留策略、上传文件配额。
