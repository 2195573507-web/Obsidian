# 软件工程学习路线

> 关联：[[软件工程/软件工程 MOC|软件工程 MOC]]

阶段：编程基础 → Web/HTTP → 数据库 → 后端框架 → 前端基础 → 测试 → 运维部署 → 架构设计 → 安全 → 产品化。

每阶段输出一个项目：CLI 工具、CRUD API、管理后台、带测试的服务、Docker 部署、CI/CD、监控告警。

检查：能读日志、能写测试、能设计表、能定位 502、能写 ADR、能做回滚。
