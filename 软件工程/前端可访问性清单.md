# 前端可访问性清单

> 关联：[[软件工程/原生前端与 UIUX 产品化实践|原生前端与 UIUX 产品化实践]]

- [ ] 按钮使用 button；
- [ ] 表单 label 关联 input；
- [ ] 图片有 alt；
- [ ] 键盘可访问；
- [ ] 焦点状态可见；
- [ ] 对比度足够；
- [ ] 错误信息可读；
- [ ] 不只依赖颜色表达状态；
- [ ] 移动端触控目标足够大。
